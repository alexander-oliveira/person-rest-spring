# Gestão de Pessoas

Projeto de aprendizado destinado à manipulação de informações pessoais através de API REST com armazenamento .

## Uso

```bash
mvnw springboot:run
```

## Endpoints

| | |
|:---:|:---|
| GET | /api/v1/people |
| GET | /api/v1/people/{id} |
| POST | /api/v1/people |
| PUT | /api/v1/people/{id} |
| DELETE | /api/v1/people |

## Tecnologias

- Java 11
- Maven 3.8.1
- Spring
- Hibernate
- H2
- MapStruct
- Lombok
